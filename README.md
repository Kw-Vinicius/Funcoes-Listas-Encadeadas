# Funcoes-Listas-Encadeadas

Fuções utilizadas:

 - Inserir Elemento na lista ( inicio / fim )
 - Exibir a lista
 - Excluir elemento da lista ( valor / posição)
 - Apagar a lista
 - Testar se existe um elemento na lista
 - Inserir elemento na posição indicada
 - Excluir elemento repetido
